Design-System Notes

Warm editorial restraint, but driven by the MycEvo contract rather than the generic starter palette. Inter/Arial/system-ui plus Chinese fallbacks keep README rendering portable.

Strengths

  • Mechanism-first flow maps directly to the README narrative.
  • Validation is visually dominant without decorative overload.
  • Candidate, reusable, and sanitized paths are distinguishable at small sizes.

Limitations

  • Dense technical labels may still need manual screenshot tuning for narrow Markdown columns.
  • The organic motif is intentionally abstract, so it reads as network governance rather than biology.
  • This is exploration-grade HTML/SVG, not final production illustration tooling.

Semantic-Fit Assessment

Strong fit for “evidence grows, governance decides”: sage connectors show accumulation, terracotta/gold gates show validation, and strict arrows prevent accidental claims of raw upload or automatic promotion.
